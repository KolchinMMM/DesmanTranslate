# DesmanTranslate
Desman Translate — веб-сервис для осуществления коллективных переводов. Пользователи могут зарегистрироваться на сайте, чтобы создавать проекты по переводу или присоединяться к существующим проектам, приглашать в свои проекты других пользователей. Пользователи загружают в проект текст произведения, который автоматически или вручную разбивается на отрывки. Участники проекта предлагают свои варианты перевода для каждого отрывка, путем голосования выбирается лучший вариант. Итоговый текст или его части можно экспортировать в желаемом формате.
